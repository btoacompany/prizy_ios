Requires python3

---------------------------
Initialize Step 
---------------------------
Type the ff:

virtualenv venv

source ./venv/bin/activate

pip install -r requirements.txt

chmod a+x *.command

---------------------------
Command Listing 
---------------------------
double click any .command to execute the script

wording.command 
	generate wording constant and .string file to proper path

segue.command
	parse all .storyboard file to list all segues 

plist.command 
	list all .plist to a file 

all.command 
	do all above
